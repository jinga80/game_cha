# 🚀 게임 변경사항 기록

## 📋 **v1.08 - 공격 시스템 및 적 AI 완전 재구현** (2024년 12월)

### ⚔️ **F키 공격 시스템 완전 재구현**
- **발사체 시스템**: 골드색 원형 발사체가 플레이어 방향으로 발사
- **폭발 효과**: 발사체가 적에게 맞으면 화려한 폭발 효과와 파티클
- **공격 애니메이션**: 공격 시 8방향 파티클과 방향별 파티클 효과
- **공격 쿨다운**: 0.33초마다 연속 공격 가능
- **발사체 클래스**: Projectile 클래스로 체계적 관리
- **폭발 효과 클래스**: Explosion 클래스로 화려한 폭발 구현

### 🧠 **적 AI 시스템 완전 구현**
- **추적 모드 (⚡)**: 플레이어가 200px 이내에 있으면 빠르게 추적하며 빨간색으로 변함
- **경계 모드 (⚠️)**: 플레이어가 400px 이내에 있으면 천천히 접근하며 노란색으로 변함  
- **순찰 모드 (🔄)**: 플레이어가 멀리 있으면 랜덤하게 이동하며 초록색으로 표시
- **AI 상태 표시**: 적 위에 이모지와 색상으로 상태 표시
- **지능적 이동**: 플레이어 방향을 바라보고 자연스럽게 추적

### 🎨 **시각적 효과 대폭 향상**
- **발사체 디자인**: 골드 몸체 + 오렌지 테두리 + 꼬리 효과
- **폭발 효과**: 20개의 다색 파티클 + 확장하는 폭발 원
- **공격 방향 표시**: 플레이어가 공격할 때 방향별 시각적 피드백
- **AI 상태 시각화**: 적의 행동 패턴을 색상과 이모지로 직관적 표시
- **향상된 파티클**: 크기 변화, 꼬리 효과, 하이라이트 효과

### 🔧 **기술적 개선**
- **클래스 기반 시스템**: Projectile, Explosion 클래스로 체계적 관리
- **충돌 감지**: 발사체와 적의 정확한 충돌 처리
- **메모리 최적화**: 화면 밖 객체 자동 제거
- **성능 향상**: 효율적인 렌더링 순서와 파티클 관리
- **코드 모듈화**: 공격 관련 로직을 별도 함수로 분리

### 🎮 **게임플레이 변화**
- **F키를 누르면** → 골드 발사체가 발사됨
- **발사체가 적에게 맞으면** → 폭발 효과와 함께 50 데미지
- **적들이 플레이어를 추적** → 거리에 따라 다른 행동 패턴
- **시각적 피드백 강화** → 모든 액션에 화려한 효과
- **전략적 게임플레이** → 적의 AI 상태를 파악하고 대응

## 📋 **v1.07 - 게임 밸런스 대폭 개선 및 스테이지 시스템 완전 구현** (2024년 12월)

### ⚔️ **적 공격 시스템 활성화**
- 적들이 실제로 플레이어를 공격하도록 완전 구현
- 적 타입별로 다른 공격력과 공격 쿨다운 시스템
- 나무돌이: 15 데미지, 나무왕: 25 데미지, 포탑몬: 20 데미지
- 적 공격 시 플레이어를 밀어내는 넉백 효과 추가

### ❤️ **체력 시스템 대폭 개선**
- 한방에 죽지 않도록 데미지량 조정 (기존 50 → 15-25)
- 무적 시간 시스템 추가 (데미지 받은 후 1초간 무적)
- 무적 상태 시 깜빡임 효과와 청록색 테두리로 시각적 표시
- 체력이 0이 되기 전까지 생명이 감소하지 않음

### 🎯 **스테이지 시스템 완전 구현**
- 스테이지 진행도 시스템 (0-100%)
- 플레이어 위치(60%) + 코인 수집(20%) + 적 처치(30%) 기반 진행도
- 스테이지 완료 시 축하 파티클과 완료 메시지 표시
- 3초 후 자동으로 다음 스테이지로 이동
- 스테이지별로 다른 적 배치와 능력치

### 📊 **스테이지 진행도 UI**
- 화면 상단 중앙에 진행도 바 표시
- 그라데이션 색상으로 진행도 시각화
- 남은 적 수와 코인 수 실시간 표시
- 스테이지 번호와 진행률 퍼센트 표시

### 🎮 **게임플레이 밸런스 조정**
- 적들의 체력과 이동 속도 조정
- 스테이지별로 점진적으로 어려워지는 난이도
- 스테이지 5부터 나무왕, 10부터 포탑몬 등장
- 적 처치 시 점수 보상 시스템

### ✨ **시각적 효과 개선**
- 무적 상태 시 깜빡임 효과
- 스테이지 완료 시 축하 파티클
- 적 타입별 색상 구분 (나무돌이: 초록, 나무왕: 진한 초록, 포탑몬: 진한 빨강)
- 진행도 바의 그라데이션 효과

### 🔧 **기술적 개선**
- 스테이지 진행도 계산 알고리즘 최적화
- 적 AI 시스템 개선 (공격 쿨다운, 화면 밖 제거)
- 메모리 사용량 최적화
- 게임 루프 성능 향상

### 🐛 **버그 수정**
- 적이 공격하지 않는 문제 완전 해결
- 한방에 죽는 문제 완전 해결
- 스테이지 종료가 명확하지 않은 문제 해결
- 게임 진행도 표시 문제 해결

### 📈 **사용자 경험 개선**
- 명확한 스테이지 진행 시스템
- 공정한 난이도 곡선
- 시각적 피드백 강화
- 직관적인 게임 진행 표시

## 📋 **v1.06 - 전체화면 시스템 및 UI 최적화** (2024년 12월)

### 🎨 **UI 시스템 대폭 개선**
- **반응형 UI**: 전체화면 모드에 따른 UI 요소 크기 자동 조정
- **미니멀 디자인**: 게임 화면을 가리지 않는 깔끔한 UI
- **동적 스타일링**: .fullscreen CSS 클래스로 전체화면 전용 스타일
- **UI 크기 최적화**: 전체화면에서 UI 요소 크기 자동 축소

### 🔧 **기술적 개선**
- **캔버스 크기 관리**: 화면 크기에 따른 자동 조정
- **이벤트 리스너**: 전체화면 상태 변경 감지 및 처리
- **크로스 브라우저 지원**: webkit, moz, ms 접두사 지원
- **윈도우 리사이즈**: 화면 크기 변경 시 자동 대응

### 📱 **사용자 경험 개선**
- **진정한 전체화면**: 게임이 화면 전체를 차지
- **직관적인 컨트롤**: 한 번의 클릭으로 전체화면 전환
- **키보드 단축키**: F11로 빠른 전체화면 전환
- **자동 UI 조정**: 전체화면에서 UI가 자동으로 최적화

## 📋 **v1.02 - 3D 시각 효과 및 밤낮 시스템** (2024년 12월)
