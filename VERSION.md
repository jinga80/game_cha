## 🚀 **v1.01 - 주요 기능 추가**

### ✨ **새로 추가된 기능**
- **2단 점프 시스템**: 공중에서도 점프 가능
- **체력 자동 회복**: 데미지 후 5초 후 자동 회복
- **스테이지 선택 시스템**: 원하는 스테이지부터 시작 가능
- **향상된 점프 메커니즘**: 이동 중에도 점프 가능

### 🔧 **기술적 개선사항**
- 점프 상태 관리 최적화
- 체력 회복 시스템 구현
- 스테이지 선택 UI 추가
- 모바일 컨트롤 반응성 향상

### 🐛 **버그 수정**
- 이동 중 점프 불가 문제 해결
- 점프 상태 초기화 문제 수정
- 플레이어 상태 관리 개선
